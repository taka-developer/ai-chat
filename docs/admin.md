# 管理画面仕様

---

## 概要

管理画面は **管理者**（STEKWIRED）と **編集者**（クライアント担当者）の 2 ロールで構成される。  
セッションベース認証。ロールによってアクセスできるページが異なる。

---

## 認証

### ログイン

```
URL: /ai-chat/admin/login.php
```

- メールアドレスとパスワードでログイン
- ログイン成功後、ロールに応じてリダイレクト
  - `admin` → `/ai-chat/admin/index.php`
  - `editor` → `/ai-chat/admin/editor/index.php`
- ログイン時に `session_regenerate_id(true)` でセッション固定攻撃を防止
- 連続 5 回失敗で 30 分ロック。ロック中は専用エラーメッセージを表示
- セッションタイムアウト（`SESSION_TIMEOUT` 秒）後はセッション破棄してリダイレクト

### ログアウト

```
URL: /ai-chat/admin/logout.php
```

セッションを破棄して `/ai-chat/admin/login.php` へリダイレクト。

## 管理者画面（STEKWIRED 専用）

### ダッシュボード `/ai-chat/admin/index.php`

| 表示項目 | 内容 |
|---|---|
| クライアント数 | 登録クライアントの総数 |
| 有効 FAQ 数 | 公開中の FAQ 総数 |
| 総会話数 | 全クライアントの累計会話数 |
| 本日の会話数 | 当日の会話数 |
| 最近の会話ログ | 直近 10 件（日時・クライアント・ユーザー入力） |

### クライアント管理 `/ai-chat/admin/clients.php`

**一覧表示**

| 列 | 内容 |
|---|---|
| 会社名 | クライアント名 |
| FAQ 数 | 有効 FAQ の件数 |
| widget_key | 埋め込みキー |
| 埋め込みタグ | コピー用 `<script>` タグ |
| 登録日 | クライアント登録日 |

**新規追加フォーム**

| 項目 | 必須 | 説明 |
|---|---|---|
| 会社名 | ✅ | クライアント名 |
| システムプロンプト | — | ボットの口調・制約 |
| お問い合わせ URL | — | フォールバック誘導先 |

追加時に `widget_key` を自動生成（`bin2hex(random_bytes(32))`）。

**インライン編集**

各行の「編集」ボタンで編集フォームを展開。会社名・システムプロンプト・お問い合わせ URL を変更可能。`widget_key` は変更不可。

### FAQ 管理 `/ai-chat/admin/faqs.php`

全クライアントの FAQ を横断的に管理する。

**CSV インポート**

| 項目 | 必須 | 説明 |
|---|---|---|
| クライアント | ✅ | インポート先クライアントを選択 |
| CSVファイル | ✅ | `.csv` 形式（UTF-8 / UTF-8 BOM 対応） |
| キーワード自動生成 | — | チェックで keywords 空行を Claude API が補完 |

対応ヘッダー（日本語・英語どちらも可）：`category` / `question` / `answer` / `keywords` / `priority`  
サンプル CSV は `/ai-chat/admin/sample_faq.csv` からダウンロード可能。

**手動追加フォーム**

| 項目 | 必須 | 説明 |
|---|---|---|
| クライアント | ✅ | 所属クライアントを選択 |
| カテゴリ | — | FAQ のカテゴリ |
| 質問 | ✅ | 質問文 |
| 回答 | ✅ | 回答文 |
| キーワード | — | 空の場合は Claude API が自動生成 |
| 優先度 | — | 数値（高いほど優先、デフォルト 0） |

**インライン編集**

各行の「編集」ボタンで編集フォームを展開。カテゴリ・質問・回答・キーワード・優先度を変更可能。キーワード空欄時は Claude API で自動生成。

**一覧操作**

- 公開 / 非公開の切り替え（即時反映）
- 削除（確認ダイアログあり）

### ユーザー管理 `/ai-chat/admin/users.php`

管理者専用。`admin_users` テーブルを管理する。

**ユーザー一覧**

| 列 | 内容 |
|---|---|
| メールアドレス | ログイン ID |
| ロール | `admin` / `editor` |
| 所属クライアント | 編集者の場合に表示 |
| 登録日 | アカウント作成日 |

**新規追加フォーム**

| 項目 | 必須 | 説明 |
|---|---|---|
| メールアドレス | ✅ | ログイン ID（重複不可） |
| パスワード | ✅ | `password_hash()` でハッシュ化して保存 |
| ロール | ✅ | `admin` / `editor` |
| 所属クライアント | editor のみ必須 | ロール選択時に動的表示 |

- 自分自身のアカウントは削除不可（誤操作防止）

### パスワード変更 `/ai-chat/admin/profile.php`

管理者・編集者共通。ヘッダーの「アカウント」リンクからアクセス。

| 項目 | 説明 |
|---|---|
| 現在のパスワード | 本人確認 |
| 新しいパスワード | 変更後のパスワード |
| 新しいパスワード（確認） | 一致しない場合はエラー |

### 会話ログ `/ai-chat/admin/logs.php`

- クライアントでフィルタリング可能
- 1 ページあたり 30 件・ページネーション付き
- 表示項目：日時、クライアント、ユーザー入力、ボット回答（80 文字抜粋）

## 編集者画面（クライアント担当者）

自社 `client_id` のデータのみ操作可能。他社データへのアクセスは不可。

### ダッシュボード `/ai-chat/admin/editor/index.php`

| 表示項目 | 内容 |
|---|---|
| 有効 FAQ 数 | 自社の公開中 FAQ 数 |
| 総会話数 | 自社の累計会話数 |
| 本日の会話数 | 当日の会話数 |
| 最近の会話 | 直近 10 件 |

### FAQ 管理 `/ai-chat/admin/editor/faqs.php`

管理者画面の FAQ 管理と同機能。ただし自社 FAQ のみ対象。

**CSV インポート**

クライアント選択不要（セッションの `client_id` を自動適用）。  
その他の仕様は管理者版と同様。

**手動追加**

- カテゴリ入力は `<datalist>` でサジェスト表示
- キーワード空欄 → Claude API で自動生成
- 公開 / 非公開の切り替え・削除

**インライン編集**

各行の「編集」ボタンで編集フォームを展開。管理者版と同機能。ただし UPDATE クエリに `AND client_id = ?` を付与し他社データの書き換えを防止。

### カテゴリ管理 `/ai-chat/admin/editor/categories.php`

`faq_categories` テーブルを管理する。

- カテゴリの追加・削除
- 追加したカテゴリは FAQ 登録時の入力候補として使用される

---

## アクセス制御

各ページの先頭で認証チェックを実施する。

```php
// 管理者専用ページ
Auth::requireAdmin();   // admin ロール以外は login.php へ

// 編集者も使用可能なページ
Auth::requireEditor();  // 未認証のみ login.php へ
```

編集者ページでの DB クエリには必ず `client_id = Auth::clientId()` の WHERE 句を付与して、他社データへのアクセスを防ぐ。

---

## セキュリティ

| 対策 | 実装 |
|---|---|
| XSS 対策 | 全出力値に `htmlspecialchars()` を適用 |
| SQL インジェクション対策 | PDO プリペアドステートメントを使用 |
| セッション固定攻撃対策 | ログイン時に `session_regenerate_id(true)` |
| 認証バイパス対策 | 全管理ページの先頭で認証チェック |
| 権限昇格対策 | 編集者クエリに `client_id` 条件を必須化 |
| ブルートフォース対策 | 連続 5 回失敗で 30 分アカウントロック |
| セッションタイムアウト | 最終アクセスから `SESSION_TIMEOUT` 秒で自動ログアウト |
